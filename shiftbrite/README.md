# Shiftbrite

This project is to have fun with shiftbrites!

## Quickstart

```sh
$ virtualenv .venv -p python2.7
$ source .venv/bin/activate
(.venv) $ pip install -r requirements.txt
(.venv) $ make clean build upload
```
